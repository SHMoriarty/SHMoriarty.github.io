Android一次踩坑记录

  • 事情经过

    昨天尝试去用activity实现一个dialog的效果，结果发现运行到这个activity时App崩溃，查看logcat发现Only fullscreen opaque activities can request orientation报错，大概意思是只有全屏不透明的activity才能设置方向，但是我之前在别的项目里使用正常，所以我去百度搜索了一下原因。发现是在Android8.0之后只有全屏不透明的activity才能设置方向，之前使用的项目都是6.0并没有到8.0所以不会报错。

  • 解决方案

    1.找到你设置透明的Activity，然后在他的theme中将android:windowIsTranslucent改为false
    <item name="android:windowIsTranslucent">false</item>
    2.再加入<item name="android:windowDisablePreview">true</item>
    3.去掉activity中的orientation属性
